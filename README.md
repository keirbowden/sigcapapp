# Signature Capture Samples
This repository contains samples (1 at the time of writing!) demonstrating how to use the Signature Capture lightning component. The Lightning Component is available on the [app exchange](https://appexchange.salesforce.com/listingDetail?listingId=a0N30000000q5XOEAY)
## Sign Here
Example Lightning application that pulls the id of the record to attach the signature to from the URL. Located at src/aura/SignHere.
